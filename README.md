### # 專案啟動 (dev)

- 設定 `.env.dev`，格式如下

```
PORT=
HOST=
MYSQL_URL=
MYSQL_HOST=
MYSQL_PORT=
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_DB_NAME=
```

- 初始化資料庫並啟動

```
yarn install && yarn run rebuild && yarn run migrate:dev && yarn run dev
```
